1. Solve by Factorization: 1/(a + b + x) = 1/a + 1/b + 1/x
Step 1: Rearrange the equation
Shift the term 1/x to the left side:
1/(a + b + x) - 1/x = 1/a + 1/b

Step 2: Take LCM on both sides
Left Side: [x - (a + b + x)] / [x(a + b + x)]
Right Side: (b + a) / ab

Simplify the numerator on the left:
(x - a - b - x) / (ax + bx + x²) = (a + b) / ab
-(a + b) / (ax + bx + x²) = (a + b) / ab

Step 3: Simplify by dividing both sides by (a + b)
-1 / (ax + bx + x²) = 1 / ab

Step 4: Cross Multiplication
-ab = x² + ax + bx
x² + ax + bx + ab = 0

Step 5: Factorizing by grouping
x(x + a) + b(x + a) = 0
(x + a)(x + b) = 0

Final Answer:
Either x + a = 0 => x = -a
Or x + b = 0 => x = -b
Solution Set = {-a, -b}
2. Show roots of (x + c)² = 4ax will be equal if c = a.
Step 1: Simplify the equation to Standard Form
Expanding $(x + c)^2$:
$x^2 + 2xc + c^2 = 4ax$

Bringing all terms to the left side:
$x^2 + 2xc - 4ax + c^2 = 0$
Taking $x$ common from the middle terms:
$x^2 + (2c - 4a)x + c^2 = 0$

Step 2: Identify Coefficients
Comparing with $Ax^2 + Bx + C = 0$:
$A = 1$
$B = (2c - 4a)$
$C = c^2$

Step 3: Condition for Equal Roots (Discriminant = 0)
Formula: $B^2 - 4AC = 0$

$(2c - 4a)^2 - 4(1)(c^2) = 0$
$(4c^2 + 16a^2 - 16ac) - 4c^2 = 0$

Step 4: Solve for c
$4c^2 - 4c^2 + 16a^2 - 16ac = 0$
$16a^2 - 16ac = 0$
Taking $16a$ common:
$16a(a - c) = 0$

As $16a \neq 0$, then:
$a - c = 0$
c = a

Hence Proved.
3. Find constant term in the expansion of (x² - 1/x)⁴
Step 1: Identify values for General Term Formula
Using formula: Tr+1 = ⁿCᵣ (a)ⁿ⁻ʳ (b)ʳ
Here: n = 4, a = x², b = -1/x (or -x⁻¹)

Step 2: Put values in the formula
Tr+1 = ⁴Cᵣ (x²)⁴⁻ʳ (-x⁻¹)ʳ
Tr+1 = ⁴Cᵣ (x⁸⁻²ʳ) (-1)ʳ (x⁻ʳ)
Combine the powers of x:
Tr+1 = ⁴Cᵣ (-1)ʳ x⁸⁻²ʳ⁻ʳ
Tr+1 = ⁴Cᵣ (-1)ʳ x⁸⁻³ʳ

Step 3: For Constant Term (Term independent of x)
The power of x must be zero:
8 - 3r = 0
3r = 8 => r = 8/3

Note: Since 'r' must be a whole number (like 0, 1, 2...), and here it is a fraction (8/3), it means there is no constant term in this specific expansion (n=4).

4. Find Middle term of (a - b)⁷
Step 1: Identify n and number of terms
Here n = 7 (odd power).
Total number of terms in expansion = n + 1 = 7 + 1 = 8.

Step 2: Locate Middle Terms
Since total terms are even (8), there are two middle terms:
1. First Middle Term = (n + 1) / 2 = (7 + 1) / 2 = 4th term
2. Second Middle Term = (n + 3) / 2 = (7 + 3) / 2 = 5th term

Step 3: Calculate 4th Term (r = 3)
Using formula: Tr+1 = ⁿCᵣ (a)ⁿ⁻ʳ (b)ʳ
T₃₊₁ = ⁷C₃ (a)⁷⁻³ (-b)³
T₄ = 35 (a⁴) (-b³)
T₄ = -35a⁴b³

Step 4: Calculate 5th Term (r = 4)
T₄₊₁ = ⁷C₄ (a)⁷⁻⁴ (-b)⁴
T₅ = 35 (a³) (b⁴)
T₅ = 35a³b⁴

Final Answer: The middle terms are -35a⁴b³ and 35a³b⁴.
5. Prove: 16 sin 20° sin 40° sin 60° sin 80° = 3
Step 1: Left Hand Side (LHS)
LHS = 16 sin 20° sin 40° sin 60° sin 80°

Step 2: Put the value of sin 60°
We know that sin 60° = √3/2
LHS = 16 × (√3/2) × [sin 20° sin 40° sin 80°]
LHS = 8√3 [sin 20° sin 40° sin 80°]

Step 3: Use the Identity
Using formula: sin θ sin(60 - θ) sin(60 + θ) = 1/4 sin 3θ
Let θ = 20°, then:
sin 20° sin(60 - 20)° sin(60 + 20)° = 1/4 sin 3(20°)
sin 20° sin 40° sin 80° = 1/4 sin 60°

Step 4: Substitute back into LHS
LHS = 8√3 × [1/4 sin 60°]
LHS = 2√3 × sin 60°
LHS = 2√3 × (√3/2)

Step 5: Final Calculation
LHS = √3 × √3 = 3

LHS = RHS (Hence Proved)
6. If sin θ = 12/13 and θ lies in 2nd quadrant, find remaining Trigonometric Ratios.
Step 1: Identify given sides
sin θ = Perpendicular / Hypotenuse = 12 / 13
So, Perpendicular (P) = 12, Hypotenuse (H) = 13

Step 2: Find Base (B) using Pythagoras Theorem
H² = P² + B²
13² = 12² + B²
169 = 144 + B²
B² = 169 - 144 = 25
Base (B) = 5

Step 3: Quadrant Check (2nd Quadrant)
In the 2nd quadrant, only Sine and Cosecant are positive.
Cosine, Secant, Tangent, and Cotangent will be negative.
Therefore, Base will be taken as -5 for these ratios.

Step 4: Remaining Ratios
1. cos θ = Base / Hyp = -5 / 13
2. tan θ = Perp / Base = -12 / 5
3. csc θ = Hyp / Perp = 13 / 12
4. sec θ = Hyp / Base = -13 / 5
5. cot θ = Base / Perp = -5 / 12
7. Prove: √[(1 + cos θ) / (1 - cos θ)] = csc θ + cot θ
Step 1: Start with Left Hand Side (LHS)
LHS = √[(1 + cos θ) / (1 - cos θ)]

Step 2: Rationalize the fraction inside the square root
Multiply and divide by the conjugate of the denominator (1 + cos θ):
= √[ {(1 + cos θ) / (1 - cos θ)} × {(1 + cos θ) / (1 + cos θ)} ]

Step 3: Simplify the Numerator and Denominator
Numerator: (1 + cos θ)(1 + cos θ) = (1 + cos θ)²
Denominator: (1 - cos θ)(1 + cos θ) = 1 - cos² θ (Using a² - b² formula)
= √[ (1 + cos θ)² / (1 - cos² θ) ]

Step 4: Use Trigonometric Identity
Since 1 - cos² θ = sin² θ, the expression becomes:
= √[ (1 + cos θ)² / sin² θ ]

Step 5: Remove the Square Root
Squares and square root cancel each other:
= (1 + cos θ) / sin θ

Step 6: Split the fraction
= 1/sin θ + cos θ/sin θ

Final Step: Apply reciprocal identities
We know 1/sin θ = csc θ and cos θ/sin θ = cot θ
= csc θ + cot θ

LHS = RHS (Proved)
8. If Position vector of A and B are 5i − 2j + 4k and i + 3j + 7k. Find the magnitude of AB and direction cosines.
Step 1: Find Vector AB
Formula: $\vec{AB} = \vec{OB} - \vec{OA}$ (Position Vector of B - Position Vector of A)
$\vec{AB} = (1i + 3j + 7k) - (5i - 2j + 4k)$
$\vec{AB} = (1-5)i + (3+2)j + (7-4)k$
$\vec{AB} = -4i + 5j + 3k$

Step 2: Find Magnitude of AB ($|\vec{AB}|$)
Formula: $\sqrt{x^2 + y^2 + z^2}$
$|\vec{AB}| = \sqrt{(-4)^2 + 5^2 + 3^2}$
$|\vec{AB}| = \sqrt{16 + 25 + 9}$
$|\vec{AB}| = \sqrt{50}$ = $5\sqrt{2}$

Step 3: Find Direction Cosines (l, m, n)
Formula: $l = x/|\vec{AB}|, \quad m = y/|\vec{AB}|, \quad n = z/|\vec{AB}|$

$l = -4 / 5\sqrt{2}$
$m = 5 / 5\sqrt{2} = \mathbf{1/\sqrt{2}}$
$n = 3 / 5\sqrt{2}$

Direction Cosines are: $(-4/5\sqrt{2}, 1/\sqrt{2}, 3/5\sqrt{2})$
