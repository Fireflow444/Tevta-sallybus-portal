2. Discuss nature of roots of 3x² + 7x + 2 = 0
Given quadratic equation:
3x² + 7x + 2 = 0

Discriminant:
D = b² − 4ac
= (7)² − 4(3)(2)
= 49 − 24
= 25
3. Find sum and product of roots of 2x² + 4 = 7x
Given equation:
2x² + 4 = 7x

Standard form:
2x² − 7x + 4 = 0

Comparing with ax² + bx + c = 0:
a = 2 , b = −7 , c = 4

Sum of roots = −b/a
Product of roots = c/a
4. Find k if roots of kx² + 4x + 3 = 0 are equal
Given quadratic equation:
kx² + 4x + 3 = 0

Condition for equal roots:
b² − 4ac = 0

(4)² − 4(k)(3) = 0
16 − 12k = 0

Convert degrees to radians:

14. Solve triangle ΔABC (γ=90°, a=250, α=40°25′)

a / sinα = b / sinβ = c / sinγ

Also, β = 90° − α (since γ = 90°)
b = a * sinβ / sinα
c = a / sinα * sinγ = a / sinα
