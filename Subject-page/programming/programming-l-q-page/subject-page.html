L/Q1. Rule for naming a variable?

Ans: In C++, there are specific rules you must follow to name a variable (also called an identifier):

1. Starting Character: The first character of a variable name must be an Alphabet or an underscore (_). It cannot start with a digit (0-9).

2. No Spaces Allowed: A variable name must be a single word. You cannot use spaces between words. To join words, use an underscore or capital letters.

3. Keywords are Prohibited: Keywords like int, float cannot be used as variable names because they have special meanings for the compiler.

4. No Special Characters: Except for the underscore (_), no special symbols like @, #, $, %, or ! are allowed in the name.

5. Case Sensitivity: C++ is case-sensitive, meaning it treats uppercase and lowercase letters differently.

Example: Result, result, and RESULT are three different variables.
