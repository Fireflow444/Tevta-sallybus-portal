S/Q:

1. What is a program?

Ans: A set of specific instructions that tells a computer exactly how to perform a particular task.

2. What is a programming language?

Ans: A formal set of rules used by programmers to write instructions for computers.

3. Difference between High and Low-level language?

Ans: High-level is human-readable, low-level is machine-oriented and directly controls hardware.

4. Difference between Assembly and Machine language?

Ans: Machine language uses binary (0s and 1s); Assembly uses human-readable mnemonics to represent those instructions.

5. What is a language translator / processor?

Ans: A software tool that converts human-written source code into machine-readable instructions for the computer.

6. Difference between Compiler and Interpreter?

Ans: A compiler translates program at once and interpreter translates and executes it line-by-line.

7. Difference between Source & Object code?

Ans: Source code is the original human-written text and object code is the translated machine-readable version.

8. What is an Assembler?

Ans: A specific translator that converts assembly language into binary machine code.

9. What is a data type?

Ans: A classification that tells the compiler what kind of data can store and process.

10. Sub-types of data-types?

Ans: Commonly divided into basic:

1. Int.

2. Char.

3. Float.

4. Arrays.

5. Pointers.

6. Structures.

11. Range of data-types?

Ans: The specific mathematical minimum and maximum values a data type can store in memory.

12. Memory, signs, and keywords?

Ans:

Data Type Name Keyword Size (Bytes) Range / Purpose

Floating Point float 4 Bytes Decimal numbers (6-7 digits)

13. What is a bug+ Types?

Ans: An error in a program is called bug. Main types are Syntax, Logical, and Runtime errors.

14. What is debugging?

Ans: The process of analysing, finding, fixing errors in a program is called debugging.

15. What is syntax?

Ans: The set of formal rules or structure and sequence of statements in a programming language.

16. What is Syntax Error?

Ans: Against the language grammatical rules, which prevents the code from compiling is called syntax error.

17. What is Logical Error?

Ans: A mistake in the program's logic that cause the incorrect result even program is running.

18. What is Runtime Error?

Ans: An error occurring during execution caused by perform illegal operations like dividing by zero.

19. What is variable?

Ans: A named memory location used to store data values that can change during program execution.

20. Rule for naming a variable?

Ans: Variable name start with a letter or underscore, cannot contain spaces, and shouldn't be a keyword.

22. What is keywords?

Ans: Reserved words with predefined meanings that the compiler understands through the language's is called keyword.
